##  What is LogicTrace AI?

LogicTrace AI isn't just another debugger—it's a **cinematic code experience**. Write Python code, hit trace, and watch your logic unfold step-by-step with beautiful visualizations, AI-powered explanations, and even auto-generated video walkthroughs.

Perfect for:
-  **Students** learning programming fundamentals
-  **Educators** teaching algorithms and data structures  
-  **Developers** debugging complex logic
-  **Anyone** who's ever wondered "what is my code actually doing?"

##  Quick Start

### Prerequisites
- Node.js 18+ 
- A [Gemini API Key](https://ai.google.dev/) (free tier available!)

### Installation

```bash
# Clone the repo
git clone https://github.com/Aakash-Lalwani/logictracer.git
cd logictracer

# Install dependencies
npm install

# Set up your API key
echo "GEMINI_API_KEY=your_key_here" > .env.local

# Launch the app
npm run dev
```

 Open [http://localhost:3000](http://localhost:3000) and start tracing!
